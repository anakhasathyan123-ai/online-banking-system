# online-banking-system
DBMS Project using PHP and MySQL
## Features

- Admin Login
- User Login
- Account Creation
- Money Transfer
- Transaction History
- Aggregate Reports
- Trigger & View Implementation

## Technologies Used

- PHP
- MySQL
- HTML
- CSS

## How to Run

1. Import the SQL file in MySQL
2. Place project folder in htdocs / server folder
3. Run login.php in browser
